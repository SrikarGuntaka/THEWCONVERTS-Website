**THEWCONVERTS is a premium e-commerce platform specializing in authentic, factory-original OEM interior and tech conversions for Ford F-150 trucks.**

Built for enthusiasts who demand perfection, THEWCONVERTS transforms base-model and single-cab trucks into luxury, high-trim builds using genuine components and precision-verified fitment.

---

## 🚀 Getting Started

To run the project locally, follow these steps:

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Development Server
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

---

## 🛠️ Tech Stack
- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite 5](https://vitejs.dev/)
- **Routing**: [React Router v6](https://reactrouter.com/)
- **Styling**: Modern Vanilla CSS (Flexbox, Grid, Glassmorphism)
- **Deployment**: Static Site Hosting (Vite-optimized)

---

## 🏗️ Project Structure
- `/src/pages`: Main application views (Home, Shop, About, etc.)
- `/src/sections`: Reusable UI sections and layout patterns
- `/src/components`: Individual UI components (Buttons, Toasts, Reveals)
- `/src/data`: Static product and configuration data
- `/src/index.css`: Global design system and custom design tokens
